Volver a Membora CRM

Politica de privacidad

Texto orientativo preparado para el TFM. Antes de un uso comercial real debe adaptarse al responsable definitivo y revisarse legalmente.

1. Responsable

Responsable del tratamiento: Jose Hurtado, como autor del proyecto academico Membora CRM. Contacto: contacto@josehurtado.dev.

2. Datos tratados

El formulario puede recoger nombre, empresa o gimnasio, email, telefono, mensaje, aceptacion RGPD, URL de origen y datos tecnicos necesarios para seguridad y trazabilidad.

3. Finalidad

Los datos se usan para gestionar solicitudes de informacion, responder al contacto comercial y registrar la solicitud en el panel Admin CRM.

4. Base legitimadora

La base principal es el consentimiento de la persona interesada al enviar el formulario y aceptar el contacto.

5. Conservacion

Los datos se conservaran durante el tiempo necesario para atender la solicitud y documentar el funcionamiento del proyecto, salvo obligacion legal o solicitud de supresion aplicable.

6. Derechos

La persona interesada puede solicitar acceso, rectificacion, supresion, oposicion, limitacion o portabilidad escribiendo al correo de contacto indicado.

7. Encargados y comunicaciones

El proyecto puede usar servicios tecnicos de hosting, base de datos y correo para operar la aplicacion. No se prevé cesion comercial de datos a terceros.
